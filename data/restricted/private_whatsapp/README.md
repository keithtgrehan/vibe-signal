# Restricted Private WhatsApp Data

This directory is for local-only private WhatsApp exports and derived private review files.

- Raw exports belong here only on a local machine, or outside the repository entirely.
- This folder is gitignored except for this README and `.gitkeep`.
- Do not commit private messages, parsed private message JSONL, participant names, screenshots, zip files, or review files containing private text.
- Consent must be documented outside the model and outside committed training artifacts.
- If a participant requests deletion, remove the raw export and all derived private files, including parsed JSONL, redacted JSONL, review CSVs, reports, and any local caches.
