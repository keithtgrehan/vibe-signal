# Closed-Beta Known Limitations

- Real-device iPhone/TestFlight QA remains pending.
- Legal review remains pending.
- Tester invites remain blocked until P0 gates pass.

- No reviewed human-label set supports accuracy or model-quality claims.

- Monitoring is manual unless upgraded.
