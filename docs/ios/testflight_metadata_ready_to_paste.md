## Beta Description

Vibe Signal is a privacy-conscious communication-support beta that highlights observable message patterns such as clarity, ambiguity, pressure, reassurance, cognitive load, and repair opportunities.

It does not infer private motives, predict attraction, detect cheating, detect lying, diagnose people, provide therapy, or optimize manipulation.

## Tester Instructions

Use synthetic examples first. Only submit messages you have permission to review. Do not submit names, phone numbers, addresses, medical/legal/financial details, workplace-sensitive content, minors' content, or highly sensitive third-party content.

Report issues with metadata only: screen name, approximate time, device, build, and a short description without private message text.
