# Feature Scope

Some users explore changes in communication patterns when trying to understand relationship dynamics.

VibeSignal AI currently scopes to:

- Conversation pattern analysis for WhatsApp exports, pasted chats, and interview/audio transcript segments
- Deterministic measurements for affect balance, uncertainty, reassurance, conflict, directness, specificity, hedging, consistency, reply timing, and hesitation markers
- Structured artifacts for messages, turns, response links, topical blocks, shift events, consistency events, and safe UI summaries
- Comparison-based outputs such as “later replies look shorter and less detailed than earlier ones”
- Optional late-bound summaries that consume only structured signals

Out of scope for this hardening pass:

- Cheating detection
- Truth or lie detection
- Pass/fail or hire/reject outputs
- Motive or hidden-intent inference
- Scraping, surveillance, or hidden cloud dependence
