# Friend A/B Feedback Questions

- What do you think this product does after 5 seconds?
- What would you click first?
- Was anything confusing before you clicked?
- Did the page make you want to try the demo?
- Did the result make sense?
- What part was most useful?
- Did anything feel creepy, too strong, or like mind-reading?
- Would you trust this with a message if you had permission to analyze it?
- Which headline works better:
  - "See what a message is doing - without guessing what someone feels."
  - "Before you reply, check what the message actually says."
  - "Read the message, not the motive."
- Which version would you rather use?
- What should be removed?
- What should be improved?
- Would you send this to a friend?
