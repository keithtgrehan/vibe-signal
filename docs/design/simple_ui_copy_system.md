# Simple UI Copy System

Date: 2026-06-04

## Core Copy

Hero: “Understand message patterns without guessing motives.”

Subcopy: “Try a synthetic demo or paste permissioned text. Vibe Signal shows observable cues, evidence, limits, and a safer next step.”

Trust line: “Observable wording only. No mind-reading or relationship verdicts.”

## Goals

- Understand what feels unclear
- Reduce pressure in my reply
- Find the direct ask
- Check if I am over-reading
- Prepare a clearer response

Goal helper: “Goal only shapes wording and suggested next steps. It does not change backend inference.”

## Modes

- Demo Mode
- Analyze Text

Demo Mode description: “Try safe synthetic examples first. No private messages needed.”

Analyze Text description: “Paste only text you have permission to review. Remove names and sensitive details.”

## Context

Label: “What kind of exchange is this?”

Helper: “Context only adjusts caution and wording. It does not infer intent.”

Options:

- General
- Relationship
- Work
- Friends/family
- Difficult conversation
- Unsure

## Analysis Style

Label: “Analysis style”

Options:

- Quick read: “Fast summary with key evidence and one next step.”
- Evidence-first: “Quoted cues, pattern labels, limits, and repair options.”
- Careful: “More cautious. Best for sensitive or unclear messages.”

Do not use “model” unless there is a real model/provider selection.

## Result Labels

- What stands out
- Evidence
- Pattern
- What this cannot tell you
- Safer next step

## Fallback And Error

Low signal: “Not enough context to read safely.”

Low-signal body: “This message is too short or context-light. Add context or try a synthetic demo.”

Error: “The backend may be waking up. Please try again in a moment.”

## Safety Rules

- No hidden-intent, cheating, attraction, lie, diagnosis, therapy, personality, attachment-style, neurotype, manipulation, compliance, or real-world accuracy claims.
- Synthetic examples are safe by default.
- Private input requires consent.
- Feedback remains metadata-only.
