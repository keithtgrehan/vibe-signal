# Delete / Export Request Process

Status: draft process. Human legal/privacy review pending.

## Intake

Use the support path approved for closed beta. Do not ask users to include raw private messages in the request.

## Locate Metadata

Search only approved systems:

- feedback metadata
- event metadata
- support records
- TestFlight/beta roster if applicable

## Delete

Delete or anonymize beta metadata according to the reviewed retention policy. Record only metadata necessary to prove completion if legal/privacy review allows it.

## Export

Export only the requester-specific metadata that legal/privacy review approves. Do not include third-party private message content.

## Escalation

Escalate immediately if the request alleges raw-message leakage, unsafe output, minor-related content, or highly sensitive third-party content.
