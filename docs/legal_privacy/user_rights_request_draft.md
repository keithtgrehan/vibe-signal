# User Rights Request Draft

Status: draft for review.

Closed-beta support should accept:

- access/export requests
- deletion requests
- correction requests where applicable
- raw-content leakage reports
- unsafe-output reports

Required metadata for lookup should avoid asking users to paste private chats. Ask for:

- tester email or TestFlight identifier if applicable
- approximate date/time
- platform
- feedback id or event id if visible
- short description without private message content

Legal/privacy owner must review final wording before beta use.
