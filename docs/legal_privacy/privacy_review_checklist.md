# Privacy Review Checklist

Status: draft checklist. Privacy review is pending.

## Data Collection

- [ ] Confirm whether raw submitted messages are stored anywhere.
- [ ] Confirm feedback stores metadata only.
- [ ] Confirm event routes store metadata only.
- [ ] Confirm logs do not include request bodies or raw messages.
- [ ] Confirm synthetic fixtures contain no real chats or tester data.
- [ ] Confirm no analytics SDKs are present.
- [ ] Confirm optional provider/BYOK scope for beta.

## User Rights / Support

- [ ] Confirm deletion request path.
- [ ] Confirm export/access request path.
- [ ] Confirm support contact.
- [ ] Confirm expected response process and owner.

## App Store / TestFlight Inputs

- [ ] Review privacy-label inventory.
- [ ] Review beta tester instructions.
- [ ] Review support, privacy, terms, and data request links.

## Non-Claims

This checklist does not establish legal compliance, GDPR compliance, EU AI Act compliance, App Store approval, or production readiness.
